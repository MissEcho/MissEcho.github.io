## 使用弹窗生成form表单

先在页面上加入一个form表单；

右键打开编辑器，更改 cCtype 为 modal

保存后即可。

```javascript
//打开模态框表单
viewmodel.communication({
    type: 'modal',
    payload: {
        mode: 'inner',
        groupCode: 'form312tg', // 设计器表单容器编号
        viewmodel: viewmodel,
    },
});
```
